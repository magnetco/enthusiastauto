# Enthusiast Auto

BMW specialist e-commerce platform.

## Quick Start

```bash
cd website && pnpm install && pnpm dev    # localhost:3000
cd studio && pnpm install && pnpm dev     # localhost:3333
cd data && pnpm install && pnpm dev       # localhost:4000
```
