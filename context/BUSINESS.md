# Business

Why the website exists, how it generates revenue, and how we measure success.

---

## Business Model

Enthusiast Auto is a BMW specialist based in Cincinnati, Ohio. The website serves three revenue streams:

| Stream | Model | Margin | Volume |
|--------|-------|--------|--------|
| **Vehicles** | Direct sales of curated BMW inventory | High | Low |
| **Parts** | E-commerce (Shopify) with fitment matching | Medium | High |
| **Services** | Lead generation for service appointments | High | Medium |

## Why the Website Exists

The website is the primary digital storefront for Enthusiast Auto, designed to:

1. **Sell vehicles** — Showcase inventory to qualified buyers nationwide
2. **Sell parts** — Provide fitment-matched parts shopping for BMW owners
3. **Generate leads** — Capture service requests and vehicle sell/consign inquiries
4. **Build authority** — Establish EAG as the go-to BMW preservation facility

## User Segments

| Segment | Goal | Primary Path |
|---------|------|--------------|
| **Vehicle Buyer** | Find and purchase a quality BMW | Homepage → Vehicles → VDP → Inquiry |
| **Parts Shopper** | Buy parts for their BMW | Homepage → Parts → Filter by vehicle → Cart → Checkout |
| **Service Customer** | Request service appointment | Homepage → Services → Request Form |
| **Seller** | Sell/consign their BMW | Homepage → Sell → Submission Form |

## Conversion Funnels

### Vehicle Sales Funnel
```
Browse Vehicles → View VDP → Submit Inquiry → Phone/Email Follow-up → Sale
```
**Conversion goal:** Inquiry form submission
**Tracking:** Form submission event, phone call tracking

### Parts E-commerce Funnel
```
Browse/Search → View PDP → Add to Cart → Checkout → Purchase
```
**Conversion goal:** Completed purchase
**Tracking:** Shopify analytics, add-to-cart events, checkout completion

### Service Lead Funnel
```
View Services → Select Service Type → Submit Request → Confirmation
```
**Conversion goal:** Service request submission
**Tracking:** Form submission event

### Vehicle Sell/Consign Funnel
```
View Sell Page → Select Option → Submit Form → Confirmation
```
**Conversion goal:** Sell submission
**Tracking:** Form submission event

## Marketing Initiatives

### Current Channels
- **Organic search** — SEO for vehicle and parts keywords
- **Direct traffic** — Brand awareness from existing customers
- **Referral** — Enthusiast forums, social shares

### Planned Channels
- **Google Ads** — Vehicle inventory campaigns
- **Social media** — Instagram, YouTube content
- **Email marketing** — Newsletter, abandoned cart recovery
- **Retargeting** — Pixel-based remarketing

### Attribution
- UTM parameters on all campaign links
- Referral source tracking on inquiry forms
